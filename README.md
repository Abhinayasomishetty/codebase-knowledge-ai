# Codebase Knowledge AI 🤖

An AI-powered tool to understand and query your codebase.

## 🚀 Features
- Ask questions about your code
- Semantic search using embeddings
- Fast retrieval with FAISS
- Supports Python projects

## 🛠️ Tech Stack
- Python
- Sentence Transformers
- FAISS
- NumPy

## 📂 Project Structure
- app.py → UI / main interaction
- main.py → core logic
- my_project/ → sample modules
- requirement.txt → dependencies

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirement.txt

2. Run the app:
   python app.py

## 💡 Example
Ask:
"How auth servers work?"

Get relevant code explanation from your project.
